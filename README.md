# Court Vision - Basketball Game Analysis System

バスケットボール試合映像を解析し、データとエビデンスに基づく指導を支援するシステム。

## Features

- 映像アップロード
- 自動スタッツ生成
- ショットチャート
- プレイリスト生成
- スカウティングレポート出力
